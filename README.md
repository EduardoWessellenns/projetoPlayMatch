# ⚽ PlayMatch

> Organize partidas. Encontre jogadores. Jogue na sua cidade.

>Isso é um projeto academico feito com a participação de varias pessoas.

PlayMatch é uma aplicação web para organização de partidas esportivas com gamificação. Qualquer pessoa pode criar ou entrar em uma partida em quadras disponíveis na sua cidade, tornando mais fácil reunir jogadores e jogar o esporte que ama.

## 🏟️ Sobre o Projeto

O **PlayMatch** facilita a organização de partidas esportivas de forma simples e gamificada. Com a plataforma, é possível:

- 📍 Visualizar quadras disponíveis na sua cidade
- 🗓️ Criar partidas em uma quadra com data, horário e esporte definidos
- 👥 Entrar em partidas criadas por outros jogadores
- 🏆 Participar de um sistema de gamificação com pontuação e conquistas

Seja futebol, basquete, vôlei ou qualquer outro esporte — o PlayMatch conecta quem quer jogar com quem precisa de jogadores.

## 🚀 Como Rodar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/EduardoWessellenns/projetoPlayMatch.git
cd projetoPlayMatch
```

### 2. Instale as dependências do backend

```bash
cd backend
npm install
```

### 3. Inicie o servidor backend

```bash
npm start
```

### 4. Abra o frontend

Acesse a pasta `frontend` e abra o arquivo `index.html` no seu navegador, ou utilize a extensão **Live Server** do VS Code.
